# Rectangle Animation App

## 開發筆記

### 主要元件：`GeometricShape.vue`

呈現 `SVG` 動畫的元件，主要使用屬性 `stroke-dasharray` 和 `stroke-dashoffset` 搭配元素`<animate>`來處理走線動畫的呈現。
